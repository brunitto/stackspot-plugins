# STACKSPOT-PLUGINS

My personal StackSpot plugins repository.

# Requirements

1. StackSpot CLI 6.

# Create new plugins

To create new plugins:

```shell
stk create plugin
```

# Validate plugins

To validate plugins (locally):

```shell
cd path/to/plugin
stk validate plugin
```

# Publish plugin

To publish plugin:

```shell
stk publish plugin --studio brunitto
```
